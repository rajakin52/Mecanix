## Phase 2 Features

1. **Invoice PDF generation** — print/download invoices as branded PDFs with workshop logo, line items, totals. Options: @react-pdf/renderer or print-friendly CSS view.

2. **Dual currency (AOA + USD)** — Angola workshops need to price in USD and accept AOA. Add secondary_currency, exchange rates, dual-currency invoices.

3. **Corporate/fleet accounts** — Consolidated billing, PO-based authorization, monthly statements for fleet operators.

4. **Vehicle inspection/check-in form** — Structured condition report (damage checklist, mileage, fuel level, personal items), damage diagram, customer signature.

5. **Appointments/booking** — Calendar-based scheduling, online booking, bay management.

6. **TecDoc parts catalogue integration** — Auto-lookup parts by vehicle.

7. **M-Pesa payment integration** — Mozambique mobile payments.

8. **WhatsApp message templates** — Meta-approved templates for automated notifications (currently using plain text).

9. **Service reminders** — Automated mileage/date-based service due reminders via WhatsApp.

10. **Mobile app polish** — Technician timer screens, customer tracking, offline sync verification.
