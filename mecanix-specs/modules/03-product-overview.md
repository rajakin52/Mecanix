# 3. Product Overview

## 3.1 Core Principles

- **Mobile-first** — mechanics update job cards from the workshop floor via tablet or phone
- **Offline-first** — full functionality without internet; syncs when connectivity restores
- **WhatsApp-native** — automated customer communication via WhatsApp Business API
- **Simple onboarding** — workshop operational in under 30 minutes
- **Lusophone-native** — pt-PT and pt-BR from day one; no translation feel
- **Ecosystem thinking** — connect workshops, customers, and insurers on one platform
- **Operationally complete** — not just job cards: covers the full business cycle including expenses, supplier bills, credit notes, and financial oversight

## 3.2 User Roles

| Role | Description | Primary Device | Access Level |
|------|-------------|----------------|--------------|
| Workshop Owner / Admin | Full system access, financial reports, settings | Desktop + Mobile | Full |
| Service Manager | Job cards, scheduling, invoicing, insurance claims | Desktop + Tablet | Operational |
| Mechanic / Technician | View & update assigned jobs, time logging, flag parts | Mobile / Tablet | Limited |
| Receptionist | Vehicle check-in, customer management, invoices | Desktop | Front-desk |
| Parts Manager | Inventory management, supplier orders | Desktop + Mobile | Inventory only |
| Customer (Vehicle Owner) | Track repairs, approve quotes, pay invoices, view history | Mobile App | Self-service |
| Insurance Assessor | Review estimates, approve/reject claims, monitor repairs | Web Portal | Insurance portal |
| Insurance Admin | Manage assessors, view analytics, configure rules | Web Portal | Insurance admin |

## 3.3 Platform Architecture Overview

MECANIX consists of four interconnected applications sharing a common backend:

1. **Workshop Management App** — The core platform for workshop staff (web + mobile)
2. **Customer App** — A lightweight mobile app for vehicle owners (iOS + Android)
3. **Technician Time Logging App** — A dedicated mobile interface optimized for mechanics on the floor
4. **Insurance Portal** — A web-based portal for insurance assessors and administrators
